# 路径查找

## 简介

亚特兰蒂斯重新现世，鉴于在人工智能方面的出众成绩，你成功成为了亚特兰蒂斯远征队的一员。在勘探队员的努力下，你们已经成功绘制并量化出了一幅亚特兰蒂斯的地图。该地图有诸多节点，而节点间由路径连接。在地形、兴趣点和资源等诸多因素的共同作用下，每条路径被启发式的赋予了一个成本/消耗值。消耗值为正则代表完成该段路程需要消耗目前的补给（收益<消耗）；消耗值为负则代表完成该路程可以获取额外的收益（收益>消耗）。

你需要负责完成一个可以高效进行最优路径规划的程序。给定地图数据、任意起始点和终点以及初始资源值后，返回：

1. 最优的路径;
2. 完成路径所需要的最少资源值；
3. 完成路径后剩余的最终资源值；
如果给定的条件无法完成该远征，则需要在此基础上额外警告无法完成。

与此同时，程序须统计以下性能指标：

## 性能指标

1. TTFQ (Time to First Query)，即从查询开始到第一个查询完成的总耗时；
2. TPQ（Time Per Query），即平均一个查询所需的时间。  

## 需提交内容
1. 整个完成上述功能的程序
2. 总结报告
